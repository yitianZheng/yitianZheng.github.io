Night's Watch暑期培训总结

（此处省略一万字……）

嗨呀，我们工作室就是不缺总结。

此篇总结采用倒叙手法~

顺德之旅

顺德之旅很好玩~逛吃逛吃~ 只是心疼自己的银子qwq……玩UNO玩出了新境界~心疼被我加了一副扑克牌的荣耀哈哈哈哈哈（每次想到都要笑死了）。第一次玩狼人玩到三点多，然后第二天睡到12点……顺德双皮奶很好吃呀，不过好像没有吃到什么其他的小吃啧啧啧……
（多图预警……果然吃货的相机的里只有吃的qwq……）

Daily CSS

为期一周的最终大任务 Daily CSS 紧赶慢赶，还是没能在出游前搞定，弄到出游前一天晚上也只完成了我负责模块的对接，真的有点遗憾。因为讨论过程中都觉得对接肯定会出很多问题，本来都已经预留了两天时间对接了，但是结果那天拖着后台过来，我们三个前端还在 debug ，真是很对不起两位后台大佬了（向后台大佬低头，顺便甩锅给设计(ノ｀Д)ノ）。不过这一星期也学到了很多东西呀，之前考核没做过登录注册这一块的内容，所以就很心水自己选的这个模块。包括用到了明文密码插件，密码强度，字符提示等等……最后对接完看到功能都完美实现实在是很开心哈哈哈。用辣鸡瀚程的话说，那一个星期天天在吃屎。（不过他整天出门左转肯定吃的比我们多ಠᴗಠ）当然虽说完成了功能，但是代码还是有很多需要改进的地方。像很多地方引用的同一套方法可以封装成函数方便调用，还有一些定位的技巧需要注意，最最重要的是，一定要多测试！一定要多测试！一定要多测试！重要的事情说三遍！！！

日常

除了最后一周，另外三周时间里都是在学习新知识。学到了很多，听红莲pt花菜momo讲课真的很有趣。学到了 git 的使用，还有如何用 hexo 方便自己写博客，还有如何创建和管理分支等等……每天都是很充实。最大的收获是一群志（yi）同（qi）道（chi）合（shi）的小伙伴啦~诞生了工作室第一个外卖男子天团哈哈哈哈哈。到最后每天每顿都在纠结外卖吃什么qwq……还有第四周经历了苦逼的智齿痛 + 头痛……不堪回首qwq……

最后

感谢这个暑假，感谢师兄师姐们，感谢小伙伴们~
以后也一起同行吧！请多多指教！
