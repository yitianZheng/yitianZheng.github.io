关于CLI的基本使用

Veröffentlicht am 2017-07-11

CLI是Command Line Interface的缩写，即命令行界面。
mkdir + 文件名：创建一个文件夹
rm : 移动文件
cd + 目录名: 进入指定目录
cd ../ : 回到当前目录上一级
ls : 查看当前目录下的所有文件
vim + 文件名：创建一个文件并进入编辑界面 >> i:进入插入模式 >> Esc:返回浏览模式 >> shift + : + w : 保存 >> shift + : + q :退出编辑界面
shift + : + wq : 保存并退出
cat + 文件名：抓取文件中的字符
pwd : 显示当前目录
