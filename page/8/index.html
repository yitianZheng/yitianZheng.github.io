关于CLI的基本使用

CLI是Command Line Interface的缩写，即命令行界面。
mkdir + 文件名：创建一个文件夹
rm : 移动文件
cd + 目录名: 进入指定目录
cd ../ : 回到当前目录上一级
ls : 查看当前目录下的所有文件
vim + 文件名：创建一个文件并进入编辑界面 >> i:进入插入模式 >> Esc:返回浏览模式 >> shift + : + w : 保存 >> shift + : + q :退出编辑界面
shift + : + wq : 保存并退出
cat + 文件名：抓取文件中的字符
pwd : 显示当前目录

写一篇新博客
hexo new + “标题” >> 在文件夹中找到相应md文件，用sublime进入编辑后保存 >> hexo g上传到本地 >> hexo d上传到服务器

git使用：分布式版本系统

建立仓库：

在个人主页中的 repositories 中新建一个仓库 >> 复制仓库链接 >> 在你想把仓库放入的文件夹里 Git Bash Here,在命令行中 git clone + 链接 >>

把你要放进仓库的文件拖入 >> 在命令行中 git add + 文件名 >> git commit -m “备注” >> git log 查看日志 >> git push origin master
